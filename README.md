# WebAR Greeting Card

A simple WebAR image-tracking demo built with [MindAR](https://github.com/hiukim/mind-ar-js) + [A-Frame](https://aframe.io).
Scan a printed card with your phone camera and see hidden content overlay.

## Stack

- MindAR image tracking (client-side compile + localStorage cache)
- A-Frame 1.5
- Static HTTPS hosting
